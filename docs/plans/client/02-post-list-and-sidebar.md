## 2. 글 목록(List) UI – 카드 그리드

### 2-1. 카드 그리드 규칙

```
.post-grid {
  display: grid;
  grid-template-columns: repeat(5, 1fr);
  gap: 20px;
}
```

**반응형 컬럼 규칙**

| 해상도 | 컬럼 |
|--------|------|
| ≥1280px | 5 |
| ≥1024px | 4 |
| ≥768px | 3 |
| ≥480px | 2 |
| Mobile | 1 |

- [ ] **4** post-grid 적용, 반응형 컬럼 breakpoint 구현. *(현재: 리스트형 flex column)*
- [x] **5** 페이지당 **5개** 노출, 페이지네이션 UI.

### 2-2. 카드 구조 (고정)

```
┌─────────────────────┐
│ Thumbnail (16:9)    │
├─────────────────────┤
│ Category            │
│ Title (max 2 lines) │
│ Summary (max 3)     │
│ Tags · Date         │
└─────────────────────┘
```

- [x] **6** Thumbnail 16:9 비율, Category, Title(max 2줄), Summary(max 3줄), Tags · Date. *(리스트형 카드: 썸네일·제목·요약·메타(카테고리·날짜))*
- [x] **7** 카드 높이 고정. *(리스트형 min-height 140px)*
- [x] **8** hover 시: translateY(-4px), shadow-lg. *(현재 -2px, shadow 적용)*

## 검색 결과
