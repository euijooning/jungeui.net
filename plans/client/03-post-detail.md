## 경로 및 데이터

- [ ] **1** 경로: `/posts/:postId`. GET /posts/{id} 로 상세 조회.
- [ ] **2** 조회수: 글 상세 페이지에 **조회수 미노출** (통계는 백오피스에서만 팔로우업).
- [ ] **3** 존재하지 않는 postId 또는 비공개 글 시 404 또는 적절한 처리.

---

## 본문 영역

- [ ] **4** 제목, 메타(카테고리·날짜·태그), 본문(content_html) 렌더링.
- [ ] **5** TOC(목차): 본문 내 헤딩 기반 목차, 클릭 시 해당 섹션으로 스크롤.
- [ ] **6** 코드 블록: 문법 강조 + Copy 버튼.
- [ ] **7** 이미지: 클릭 시 라이트박스(확대) 표시.
- [ ] **8** 유튜브 임베드: 본문 내 유튜브 URL/노드 정상 재생.
